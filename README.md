# Sistema de Gimnasio 

CRUD desarrollado con:
- Java 22
- Spring Boot
- MySQL
- JPA / Hibernate

## Funcionalidades

- Alta/Baja/Modificacion de clientes
- Gestion de membresias
- API REST

## Como correrlo

1. Configura `application.properties` con tu base de datos
2. Ejecuta el proyecto con Maven o desde tu IDE
